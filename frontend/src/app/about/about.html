About Interview Platform

We help teams run fair, structured technical interviews with less hassle and more signal.

Our mission

Make interviews human, focused, and data-driven. The platform combines real-time collaboration with structured note-taking and AI summaries to reduce bias and speed up decisions.

1) Live session

Start a room, share a link, and talk face-to-face via WebRTC.

2) Code together

Use the shared editor to solve tasks while you capture structured notes.

3) Decide

Review notes & auto-summaries; recommend with confidence.

Tech & Security

  • Frontend: Angular 17+, Tailwind CSS
  • Backend: Django + DRF, JWT in HttpOnly cookies
  • WebSocket: Node.js + Socket.IO
  • DB: PostgreSQL • Deployment: Docker + NGINX
  • Private interview rooms with role-based access; per-interviewer notes
